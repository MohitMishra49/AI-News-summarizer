# 📰 T5 News Summarizer

A deep learning-based News Summarization system built using Google's T5 (Text-To-Text Transfer Transformer) model and fine-tuned on a news summarization dataset. The model generates concise, human-readable summaries from lengthy news articles while preserving key information and context.

## ⚙️ Model Training

The T5 model was fine-tuned on a news summarization dataset to learn abstractive summarization. Unlike extractive methods, the model generates new sentences that capture the meaning of the original article.

### Training Workflow

1. Load and preprocess news articles
2. Tokenize articles and target summaries
3. Fine-tune pre-trained T5 model
4. Evaluate performance
5. Save model and tokenizer
6. Deploy using Gradio and Hugging Face Spaces

---

## 🎯 Usage
